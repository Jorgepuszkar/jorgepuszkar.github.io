# Jorge Puszkar · Portafolio Mundial

Portafolio personal interactivo construido con HTML, CSS y JavaScript vanilla. Desplegado en GitHub Pages sin ningún proceso de compilación.

## Descripción

El sitio tiene dos modos que se pueden alternar con un botón:

**Modo Mapa** — Vista principal con un mapa interactivo oscuro (estilo centro de misiones) con fotos aéreas de drone ancladas a sus ubicaciones geográficas reales. Al hacer clic en un marcador se abre un panel lateral con la sección correspondiente de mi currículum.

**Modo Clásico** — Currículum profesional tradicional con diseño limpio, listo para imprimir.

También incluye una galería de fotografía aérea con lightbox.

## Tecnologías

- [Leaflet.js](https://leafletjs.com/) — mapa interactivo con tiles de CartoDB Dark Matter
- HTML5, CSS3, JavaScript vanilla
- Compatible con GitHub Pages (sin dependencias ni proceso de build)
